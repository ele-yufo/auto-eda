# 数据叙事框架

> 有了数据发现，还要会"讲"出来。管理者不需要统计细节，需要的是：一个关键数字 + 一个故事 + 一个行动建议。

---

## 核心原则

1. **"一个可引用的关键数字"** — 每个分析最终要精炼为一个用户可以立刻引用给老板的数字
2. **不说"p-value < 0.05"** — 说"这个差异是可靠的，不是偶然的"
3. **不说"R² = 0.85"** — 说"这个模型能解释 85% 的变化，预测准确率较高"
4. **每张图表必须附带 "So What?"** — 不是"这张图显示了 X 和 Y 的关系"，而是"这意味着如果你增加 X，Y 可能会…"

---

## 框架 1：SCQA（推荐用于诊断型分析报告）

| 步骤 | 含义 | 示例 |
|------|------|------|
| **S** (Situation) | 情境：当前的客观现状 | "过去 12 个月，我们的客户复购率一直维持在 35% 左右" |
| **C** (Complication) | 冲突：打破现状的变化 | "但最近 3 个月，复购率骤降到 22%，损失了约 200 万营收" |
| **Q** (Question) | 问题：由冲突引发的核心问题 | "为什么复购率突然下降？哪些客户群受影响最大？" |
| **A** (Answer) | 答案：数据给出的回答 + 行动建议 | "数据显示新推出的会员体系反而降低了中等价值客户的复购意愿。建议…" |

**适用场景**：管理者带着一个问题来找你，需要根因分析。

---

## 框架 2：三幕式（推荐用于探索型分析报告）

| 幕 | 内容 | 占比 |
|---|------|------|
| **第一幕：挑战** | 展示数据背景 + 关键指标现状 + 核心发现 | 20% |
| **第二幕：发现** | 深入分析 → 模式 → 根因 → 反直觉发现 | 50% |
| **第三幕：行动** | 数据支持的建议 + 优先级排序 + 风险提示 | 30% |

**适用场景**：用户没有明确问题，希望从数据中发现价值。

---

## 框架 3：AIDA（推荐用于说服管理层采纳建议）

| 步骤 | 目的 | 技巧 |
|------|------|------|
| **A** (Attention) | 抓住注意力 | 用一个惊人的数字开场："我们每月因 XX 损失 50 万" |
| **I** (Interest) | 引发兴趣 | 展示数据发现，特别是反直觉的发现 |
| **D** (Desire) | 激发意愿 | 展示可能的改善效果："如果改进，预计可以…" |
| **A** (Action) | 推动行动 | 给出具体的、可执行的 Quick Win |

**适用场景**：需要说服管理层投入资源。

## 术语翻译指南

| 统计语言 ❌ | 业务语言 ✅ |
|---|---|
| p-value < 0.05 | 这个差异是可靠的，不太可能是偶然的 |
| p-value > 0.05 | 数据暂时无法证明有差异，但不代表没有差异 |
| 相关系数 0.7 | 两个指标有较强的正向关联 |
| 中位数 vs 均值 | 典型值 vs 平均值（平均值容易被极端值拉高） |
| 标准差 | 数据的波动幅度 |
| 95% 置信区间 | 真实值很可能在这个范围内 |
| 回归系数 β=2.3 | 每增加 1 单位 X，Y 平均增加 2.3 单位 |
| R² = 0.65 | 模型能解释 65% 的变化，还有 35% 受其他因素影响 |
| 过拟合 | 模型把过去的特殊情况也学进去了，预测新数据时可能不准 |
| 特征重要性 | 影响结果的关键因素排名 |
| SHAP 值 | 每个因素对最终结果的具体贡献 |
| 聚类 | 把相似的客户/产品/事件自动归类分组 |
| 异常值 | 和大部分数据差异很大的特殊个案 |
| 偏度 | 数据的分布是否偏向一侧（左偏/右偏） |
| 多重共线性 | 几个因素之间太像了，影响了分析准确性 |

## 可视化叙事技巧

### 1. 先结论后证据
```
❌ "请看下面这张散点图..."
✅ "高客单价用户的复购率是低客单价用户的 2.3 倍。以下是数据支持："
```

### 2. 对比产生洞察
```
❌ "A 组的转化率是 15%"
✅ "A 组的转化率（15%）是 B 组（8%）的近 2 倍，这意味着..."
```

### 3. 趋势胜于快照
```
❌ "当前 ARPU 是 45 元"
✅ "ARPU 从 6 个月前的 52 元下降到现在的 45 元，下降了 13%"
```

### 4. 异常值是故事的起点
```
❌ "发现 3 个异常值"
✅ "有 3 个区域的客户满意度远低于平均水平（> 2个标准差），值得重点关注"
```

### 5. 不确定性是诚实的表现
```
❌ "模型预测下个月销售额是 132 万"
✅ "模型预测下个月销售额在 118-146 万之间（95% 置信区间），中位预测 132 万"
```
